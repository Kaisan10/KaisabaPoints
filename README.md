# かい鯖グループポイント プラグイン

かい鯖グループポイントを利用して、サーバー内でショップの作成・購入を行えるプラグインです。

---

## セットアップ手順

1. https://points.bac0n.f5.si/settings へアクセスします。
2. サービスアカウントを作成します。
3. APIキーをコピーします。
4. プラグインをサーバーに導入し、一度起動します。
5. `plugins/KaisabaPoints/config.yml` を開きます。
6. `api-key: "<ここ>"` にコピーしたAPIキーを貼り付けます。
7. サーバーを再起動します。

---

## ショップの作り方

### 1. Webでの事前準備
1. https://points.bac0n.f5.si/settings へアクセスします。
2. 「商品管理」から追加したいアイテムを登録し、割り振られたアイテムIDを控えておきます。

### 2. ゲーム内での設定
1. コマンドを実行してショップを作成します。
   `/shop create <ショップID> <ショップ名>` （※ショップIDに日本語は使用不可）
2. 左側の「編集モード切替」を ON にします。
3. 中央の「アイテム設定」を開きます。
4. ショップで販売したいアイテムを配置します（削除したい場合は対象アイテムをクリック）。
5. 左下の「IDモード」を ON にします。
6. 設定したアイテムをクリックし、該当するアイテムIDの数値と同じ個数だけ空きスロットへ配置します（※ID数を減らす場合はコンパスをクリック）。
7. 左下の「戻る」ボタンを2回押します。
8. 右側の「在庫管理」を開き、必要な分だけ在庫を補充します。
9. 「編集モード切替」を OFF にして完了です。

---

## ショップでの購入手順

1. `/shop` を実行します。
2. 開いたGUIメニューから利用したいショップを選択します。
3. 購入したいアイテムをクリックします。
4. 確認画面で OK を選択すると購入が完了します。

---

## ショップの削除方法

1. `/shop delete <ショップID>` を実行します。
2. 確認画面で OK を選択すると削除が完了します。
